# Operating Systems Cache & IPC

A low-level systems project focused on memory management, cache structures, and Inter-Process Communication (IPC).

## 🛠️ Technologies & Tools
- **Language:** C
- **Concepts:** Memory Management, Cache policies (LRU), IPC, Shell scripting

## 🚀 Key Implementation
- Developed low-level cache structures with LRU (Least Recently Used) replacement policies in C.
- Implemented Inter-Process Communication (IPC) mechanisms for server-client architecture coordination.
- Built automated scripts to evaluate cache size impact and compare parallel vs sequential searches.
